# Claude Cowork — Lightning Lessons

![Cowork processing](images/banner.png)

Hands-on labs for **Claude Cowork**: set up the desktop app, run product-management workflows (job search, market research, presentations), and build a visual dashboard. By the end you’ll use Cowork for real PM tasks with files, web search, and clear outputs.

---

## Quick links to modules

| Module | What you’ll do |
|--------|----------------|
| **[Module 0: Getting Started](<Module 0 : Getting Started/0.0-introduction.md>)** | Course overview and installation. Learn what Cowork is, install the Claude Desktop app on your Mac, enable Cowork, and choose a working folder so you’re ready for the rest of the labs. |
| **[Module 1: PM–Cowork hands-on](<Module 1 : PM-Cowork-HandsOn/1.0-cowork-in-action.md>)** | Product-manager workflows in Cowork. Use Cowork as a job searcher (resume + filters), run market research with context files and web search, and turn company docs into a PM-ready presentation. |
| **[Module 2: Creating a visualizer](<Module 2 : Creating-Visualizer/creating-dashboard.md>)** | Build a dashboard with Cowork. Create a project folder, give Claude a channel link and goal, and let Cowork (with the browser extension) collect data and generate a progress dashboard you can open in a browser. |
